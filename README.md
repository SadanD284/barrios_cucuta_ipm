# Barrios San José de Cúcuta

Este repositorio contiene la digitalización cartográfica de los barrios en la ciudad de San José de Cúcuta, Norte de Santander. La construcción de los polígonos se hace a partir del Plan de Ordenamiento Territorial (POT) en lo establecido por el Acuerdo No. 022 de 2019. El objetivo principal de la iniciativa es construir unidades geográficas editables y disponibles para la investigación, la academia y la sociedad civil, siendo particularmente útil en estudios urbanos. Las capas almacenadas en este repositorio no constituyen una cartografía oficial del municipio, sino una reconstrucción digital para fines analíticos.

## Objetivo del dataset

- Construir una capa geográfica editable de barrios de San José de Cúcuta.
- Generar una unidad espacial homogénea para análisis estadístico y econométrico.
- Facilitar la estimación de indicadores sociales a escala intraurbana.
- Proveer una base reproducible para diferentes tipos de estudios.
